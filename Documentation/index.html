Getting Started

Up and running in under 60 seconds — no build tools required.

Running Locally

The template lives entirely inside the dist/ folder as self-contained HTML. No Node.js, no build step.

1
Open the folder

Open the unzipped kinetic-fitness/ root in VS Code or your editor of choice.

2
Navigate to dist/

All production files are bundled and ready inside the dist/ directory.

3
Launch in browser

Open dist/index.html directly, or use the Live Server VS Code extension for hot-reload.

Deploying

Upload the entire dist/ folder to any static host. Zero server-side config required.

Changing Theme Colors

Open dist/assets/index-*.css and update the :root block at the top of the file:

:root {
  --color-primary:    #YOUR_BRAND_COLOR;
  --color-secondary:  #YOUR_ACCENT_COLOR;
  --color-background: #YOUR_BG_COLOR;
}

Replacing Images

All images are located in dist/images/. Replace files with the same filename to swap them out without touching any code.

💡 Tip: Use 16:9 or 4:3 aspect ratios for hero sections, and square crops for testimonial/team headshots.

Fonts

The template uses Epilogue (headings) and Manrope (body) from Google Fonts. To change fonts, update the <link> tag in index.html and the --font-heading / --font-body CSS variables.

Adjusting Animation Speed

Find the GSAP timeline in assets/index-*.js. Tweak duration and ease values:

// Scroll-driven section entrance
gsap.from(".section-element", {
  opacity: 0,
  y: 40,
  duration: 1.2,     // ← slower = more cinematic
  ease: "power4.out",  // ← try "expo.out" for snappiness
  stagger: 0.08,
  scrollTrigger: {
    trigger: ".section-element",
    start: "top 85%"
  }
});

Disabling Smooth Scroll

To disable Lenis (e.g., for accessibility or performance), remove or comment out the Lenis initialization block in index.html:

// Remove or comment this block to disable smooth scroll:
// const lenis = new Lenis();
// function raf(time) { lenis.raf(time); requestAnimationFrame(raf); }
// requestAnimationFrame(raf);
