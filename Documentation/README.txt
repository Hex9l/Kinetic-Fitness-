KINETIC FITNESS - PREMIUM LANDING PAGE TEMPLATE
========================================================================

Thank you for choosing Kinetic Fitness! This premium template is designed
to be high-performance, visually stunning, and easy to deploy.

QUICK START GUIDE
-----------------
1. Open the "dist" folder.
2. Open "index.html" in your browser to view the template.
3. To customize, refer to the full documentation in the same folder.

PROJECT STRUCTURE
-----------------
/dist             - Production files (deploy these to your server)
  /assets         - CSS and JavaScript (Minified/Bundled)
  /images         - Template images (swap these with your own)
  index.html      - The main website file
/Documentation    - Full setup and customization guide
README.txt        - This file

CUSTOMIZATION
-------------
- Root colors: Located in the /assets/index-*.css file under :root.
- Images: Replace files in /dist/images/ with same filenames.
- Text: Edit content directly in /dist/index.html.
