QUICK START GUIDE
--------------------------------------------------------------------------------

1. Unzip the package to your desired location.
2. Open the 'dist' folder.
3. Launch 'index.html' in your preferred web browser to view the template.
4. To customize the content, open 'index.html' in a code editor (like VS Code).
5. Detailed instructions can be found in the 'Documentation' folder.
